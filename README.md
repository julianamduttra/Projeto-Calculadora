# Projeto: Calculadora de Risco Financeiro

Este repositório contém ferramentas iniciais para automação de cálculos de risco.

## 1. Script Shell (calculadora.sh)
Este arquivo automatiza a execução de cálculos simples diretamente no terminal Linux (Ubuntu/WSL).

**Como executar:**
1. Garanta que o arquivo tenha permissão de execução: `chmod +x calculadora.sh`
2. Execute o script com o comando: `./calculadora.sh`

## 2. Código Python (calculadora.py)
O código em Python realiza a lógica de cálculo de forma estruturada. 
- Ele utiliza a função `input()` para receber dados do usuário.
- Realiza o processamento aritmético.
- Exibe o resultado final, servindo de base para futuros sistemas de score de crédito.
